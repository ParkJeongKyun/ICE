자주 묻는 질문(FAQ)

Q. 웹 기반 헥스 뷰어(Hex Viewer)는 어떤 기능을 제공하나요?

파일의 원본 바이너리 데이터를 16진수로 시각화하여 정밀 분석을 돕습니다. 특정 오프셋(Offset) 즉시 이동은 물론, 데이터를 UInt, Float, Time 등 다양한 타입으로 자동 해석하여 전문적인 포렌식 분석 환경을 제공합니다.

Q. 이미지 파일만 분석할 수 있나요?

아닙니다. EXIF 및 청크 분석 기능은 이미지 파일에 최적화되어 있지만, 헥스 뷰어(Hex Viewer)와 해시(Hash) 계산기는 실행 파일(EXE), 문서, 압축 파일 등 모든 종류의 바이너리 파일을 지원합니다.

Q. 파일 내에서 특정 문자열이나 16진수 값을 검색할 수 있나요?

네, 텍스트(ASCII), 16진수(Hex), 오프셋(Offset) 기반의 강력한 검색 기능을 지원합니다. 대소문자 구분 옵션을 제공하며, 대용량 파일에서도 원하는 데이터를 즉시 찾아 최대 1,000개의 결과를 빠르게 탐색할 수 있습니다.

Q. 지원하는 파일 해시(Hash) 알고리즘 종류는 무엇인가요?

파일의 동일성 및 무결성 검증을 위해 MD5, SHA-1, SHA-256, SHA-512 등 표준 해시 알고리즘을 모두 지원합니다. 대용량 파일도 로컬 WASM 기술을 통해 지연 없이 즉석에서 계산됩니다.

Q. 대용량 이미지나 바이너리 파일도 분석할 수 있나요?

네, WebAssembly(WASM) 기반의 효율적인 메모리 관리 기술을 사용하여 수백 MB 이상의 대용량 파일도 브라우저 멈춤 현상 없이 실시간으로 로드하고 분석할 수 있습니다.

Q. 내 사진이나 분석 데이터가 서버에 저장되지는 않나요?

네, 원본 사진이나 파일 데이터는 서버에 절대 저장되지 않습니다. 파일 분석은 100% 사용자 기기(브라우저) 내에서 로컬로 처리됩니다. 단, 사용자가 명시적으로 '지도 보기'나 '내 IP 조회' 기능을 요청할 때만 해당 기능 수행을 위해 필요한 최소한의 데이터(GPS 좌표, IP)가 외부 API(OpenStreetMap 등)로 일회성 전송되며, 이 역시 당사 서버에 저장되지 않습니다.

Q. 생성형 AI(Stable Diffusion, NovelAI)로 만든 이미지의 프롬프트 추출이 가능한가요?

네, 대부분의 경우 추출이 가능합니다. PNG 파일 내 iTXt, tEXt 청크에 기록된 프롬프트, 모델 정보, 시드값(Seed) 등의 텍스트 데이터를 스캔하여 보여줍니다. WebUI, ComfyUI, NovelAI 등 주요 플랫폼에서 생성된 메타데이터를 확인할 수 있으나, 이미지 생성 방식이나 압축/변환 과정에 따라 일부 정보가 누락될 수도 있습니다.

Q. 파일이 손상되어 열리지 않는 이미지도 분석이 가능한가요?

네, 가능합니다. 일반적인 이미지 렌더링 방식이 아닌 바이너리 스트림 스캔 방식을 사용하기 때문입니다. 파일 헤더(IHDR)나 메타데이터 청크가 남아 있다면 깨진 이미지에서도 텍스트 정보를 복원해냅니다.

Q. 스마트폰(아이폰, 안드로이드) 스크린샷 파일도 지원하나요?

네, 모바일 기기 캡처본에 포함된 기기 정보, 캡처 시각, 해상도 등 eXIf 및 PNG 표준 청크 데이터를 완벽하게 분석하여 상세 정보를 제공합니다.

Q. 윈도우(Windows) 외에 맥(macOS)이나 모바일에서도 사용할 수 있나요?

네, 웹 브라우저가 실행되는 모든 기기(Windows, macOS, Linux, iOS, Android 등)에서 별도의 프로그램 설치 없이 즉시 사용할 수 있습니다. 인터넷만 연결되어 있다면 언제 어디서나 강력한 포렌식 환경을 제공합니다.

Q. 여러 개의 파일을 동시에 열어서 비교하거나 분석할 수 있나요?

네, 다중 탭(Multi-tab) 기능을 지원하여 여러 파일을 동시에 열어두고 번갈아 가며 분석할 수 있습니다. 또한, 한 번 계산된 해시값이나 검색 결과는 브라우저에 자동 캐시(Cache)되어 재조회 시 지연 없이 즉각적으로 표시됩니다.

활용 사례

• 파일 해시(MD5, SHA-256 등) 계산을 통한 무결성 검증 및 데이터 위변조 확인
• AI 생성 이미지(Stable Diffusion 등)에 숨겨진 프롬프트 및 파라미터 복원
• 디지털 사진 내 GPS 위치 정보 및 촬영 상세 메타데이터(EXIF) 확인
• SNS 공유 전 사진에 포함된 개인정보 유출 가능성 점검 및 프라이버시 보호
• PNG 파일 구조 정밀 분석을 통한 스테가노그래피 및 은닉 데이터 탐지
• 보안 전문가를 위한 실시간 바이너리 Hex 분석 및 오프셋(Offset) 추적
• 이미지 압축 방식(Deflate) 및 인터레이스 설정 검증을 통한 웹 최적화
• 서버 전송이 없는 안전한 로컬(Local) 환경에서 의심스러운 파일(악성코드 등)의 헤더 및 시그니처 분석
• 컴퓨터 공학 및 정보 보안 전공자를 위한 파일 구조 파악 및 리버스 엔지니어링 실습
• Data Inspector 기능을 활용한 엔디안(Little/Big Endian) 변환 및 다양한 데이터 타입(Int, Float, Time 등) 즉각 검증
• 인터넷 환경에서의 즉각적인 디지털 포렌식 수행
• 다중 탭(Multi-tab) 기능을 활용한 정상 파일과 변조 의심 파일 간의 바이너리(Hex) 실시간 비교 분석

사용 방법

파일 열기

  • 상단의 Open 버튼을 클릭하거나 파일을 드래그앤드롭하세요.
  • 여러 파일을 탭으로 동시에 열어서 분석할 수 있습니다.

분석 결과

이미지 파일

  • Thumbnail: 이미지를 미리 볼 수 있습니다.
  • EXIF Data: 카메라 정보, 촬영 일시 등 메타데이터를 확인할 수 있습니다.
  • Map: GPS 좌표가 있으면 촬영 위치를 지도에서 확인할 수 있습니다.

모든 파일

  • File Info: 파일명, 크기, 타입, 수정일 등을 확인할 수 있습니다.
  • Hash Calculator: SHA256, SHA512, MD5, SHA1 해시를 계산할 수 있습니다.
  • Hex Viewer: 파일을 16진수로 볼 수 있습니다.

검색

  • Hex: FF D8 FF처럼 16진수로 검색할 수 있습니다.
  • ASCII: JFIF처럼 문자열로 검색할 수 있습니다.
  • Offset: 0x1000처럼 특정 위치로 이동할 수 있습니다.
  • PREV/NEXT 버튼으로 결과를 이동하고, X 버튼으로 초기화할 수 있습니다.

기타

  • 상단 메뉴에서 언어를 전환할 수 있습니다.
  • 검색 결과와 해시 값은 자동으로 캐시되어 빠르게 재사용할 수 있습니다.
